Front-end Style Guide
=====================
## CSS
##### Using every declaration just once.
*Bad:*
```
h1 { color: black; }
h2 { color: black; }
h3 { color: black; }
p { color: black; }
```
*Good:*
```
h1, h2, h3, p { color: black; }
```
*Best (if possible):*
```
body { color: black; }
```
---
Don’t concatenate ID’s or Classnames with tag names.
